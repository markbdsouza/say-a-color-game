Say a Color Game
Instructions : You have 30 seconds after clicking Start, to call out as many colors as you possibly can. This game uses Chrome's Speech Recognition which is an Experimental Web Platform feature. Hence, it might not be able to read every single word you say perfectly.
Set up :
1) You need to be using the Chrome Browser
2) Type "about:flags" in the url and make sure you have "Experimental web platform features" enabled
